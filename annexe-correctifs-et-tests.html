CashDemandController — GET /cash cassé

Le problème : la méthode getAll() déclarait un paramètre @PathVariable UUID id alors que la route GET /cash n'a pas de variable de chemin ⇒ Spring levait une erreur à chaque appel de la liste (500).

Ce que ce fichier corrige aussi : injection par constructeur au lieu de @Autowired sur champ, méthodes renommées en camelCase (cancelRequest, cancelOperations), et suppression du doublon GET /getCashDemandById (la route RESTful GET /cash/{id} fait la même chose).

Attention Si le front (ou un collègue) appelle déjà GET /cash/getCashDemandById?cashRequestId=..., préviens-le de basculer sur GET /cash/{id} — ou dis-le-moi et je te redonne le fichier avec le doublon conservé.

Correctif 2 · I-02 — Bloquant

CashOperation — la règle R6 (ventilation) était buguée

Le problème : la ligne faceValuesMap.get(opDenom.getDenomination()) cherchait un objet Denomination comme clé dans une Map indexée par libellé (String). Une clé du mauvais type ne matche jamais ⇒ le résultat était toujours null ⇒ chaque création d'alimentation de caisse avec ventilation levait « cupure inconu », même avec des coupures parfaitement valides.

Ce que la nouvelle version fait :

  • Recherche par getDenominationLabel() (la vraie clé de la Map) ;
  • La méthode utilise this au lieu de se recevoir elle-même en paramètre (l'ancienne signature validateAndCalculateTotal(CashOperation op, Map...) devient validateAndCalculateTotal(Map...) — c'est pour ça que le Correctif 3 met à jour l'appelant) ;
  • Vérifie la cohérence du total déclaré : si le client annonce 250 000 MAD mais que Σ (valeur faciale × quantité) donne autre chose ⇒ erreur explicite ;
  • Normalise : le total prouvé par la ventilation est toujours stocké dans totalAmountByCurrency ;
  • Contrôle les cas dégénérés : quantité négative ou absente, ligne sans coupure ;
  • Supprime les imports jakarta.persistence (le domaine doit rester pur — I-08).

package com.bmci.q360.domain.entity;

import com.bmci.q360.domain.entity.enums.OperationType;
import com.bmci.q360.domain.entity.enums.RequestType;
import com.bmci.q360.domain.valueobjects.CashOperationId;
import lombok.AllArgsConstructor;
import lombok.Getter;
import lombok.NoArgsConstructor;
import lombok.Setter;

import java.math.BigDecimal;
import java.time.LocalDateTime;
import java.util.ArrayList;
import java.util.List;
import java.util.Map;

/**
 * @author Hamza Marouane
 * Correctifs I-02 + I-08 :
 *  - validateAndCalculateTotal : la recherche de la valeur faciale utilisait
 *    l'objet Denomination comme cle d'une Map indexee par libelle (String)
 *    => toujours null => "cupure inconu" a chaque creation. Corrige avec
 *    getDenominationLabel(). La methode utilise desormais "this" au lieu de
 *    se recevoir elle-meme en parametre, verifie la coherence du total
 *    declare (regle R6) et normalise le total calcule.
 *  - suppression des imports jakarta.persistence (purete du domaine).
 */
@AllArgsConstructor
@NoArgsConstructor
@Getter
@Setter
public class CashOperation {

    private CashOperationId id;

    private CashRequest cashRequest;

    private Currency currency;

    private OperationType operationType;

    private Integer institutionCode = 1;
    private String cashDeskNumber;
    private String isUrgent;
    private String targetCashDeskNumber;
    private RequestType operationSens;

    private LocalDateTime createdAt;
    private LocalDateTime updatedAt;
    private String operationStatus;
    private String establishment;

    private List<OperationCurrencyDetail> currencyDetails = new ArrayList<>();

    /**
     * Regle R6 : pour une alimentation de caisse (SUPPLY + CASH_DESK),
     * la ventilation par coupures est obligatoire et le total de chaque
     * devise doit valoir la somme (valeur faciale x quantite) de ses lignes.
     *
     * @param faceValuesMap referentiel { libelle de coupure -> valeur faciale }
     */
    public void validateAndCalculateTotal(Map<String, BigDecimal> faceValuesMap) {
        if (this.operationSens != RequestType.SUPPLY || this.operationType != OperationType.CASH_DESK) {
            return; // regle applicable uniquement aux alimentations de caisse
        }

        if (this.currencyDetails == null || this.currencyDetails.isEmpty()) {
            throw new IllegalStateException(
                    "Les details des coupures sont obligatoires pour alimenter une caisse");
        }

        for (OperationCurrencyDetail detail : this.currencyDetails) {
            String devise = (detail.getCurrency() != null && detail.getCurrency().getIsoCode() != null)
                    ? detail.getCurrency().getIsoCode()
                    : "?";

            if (detail.getOperationDenominations() == null || detail.getOperationDenominations().isEmpty()) {
                throw new IllegalStateException(
                        "La ventilation par coupures est obligatoire pour la devise " + devise);
            }

            BigDecimal totalCalcule = BigDecimal.ZERO;
            for (OperationDenomination ligne : detail.getOperationDenominations()) {

                if (ligne.getDenomination() == null
                        || ligne.getDenomination().getDenominationLabel() == null) {
                    throw new IllegalArgumentException(
                            "Ligne de ventilation sans coupure pour la devise " + devise);
                }

                String label = ligne.getDenomination().getDenominationLabel();

                // FIX I-02 : la cle est le LIBELLE, pas l'objet Denomination
                BigDecimal faceValue = faceValuesMap.get(label);
                if (faceValue == null) {
                    throw new IllegalArgumentException("Coupure inconnue : " + label);
                }

                if (ligne.getQuantity() == null || ligne.getQuantity() < 0) {
                    throw new IllegalArgumentException("Quantite invalide pour la coupure " + label);
                }

                totalCalcule = totalCalcule.add(
                        faceValue.multiply(BigDecimal.valueOf(ligne.getQuantity())));
            }

            // Si un total a ete declare par le client, il doit etre coherent
            if (detail.getTotalAmountByCurrency() != null
                    && detail.getTotalAmountByCurrency().compareTo(totalCalcule) != 0) {
                throw new IllegalStateException("Total incoherent pour la devise " + devise
                        + " : declare = " + detail.getTotalAmountByCurrency()
                        + ", calcule = " + totalCalcule);
            }

            // Normalisation : on stocke toujours le total prouve par la ventilation
            detail.setTotalAmountByCurrency(totalCalcule);
        }
    }
}

/**
 * @author Hamza Marouane
 * Correctifs :
 *  - create() : reactivation de la regle R1 (au moins une operation) via
 *    cashRequest.validateRequest(), et appel de la nouvelle signature
 *    op.validateAndCalculateTotal(faceValuesMap) — voir CashOperation.
 *  - update() : suppression du .toString() sur getRequestStatus() qui
 *    provoquait un NullPointerException quand le statut etait absent
 *    ("CANCELLED".equals(null) renvoie simplement false).

@Slf4j
@RequiredArgsConstructor
@Service
public class CashDemandUseCaseImpl implements CashDemandUseCase {

    private final CashDemandRepository cashDemandRepository;
    private final DenominationRepository denominationRepository;
    private final CashRequestManualMapper cashRequestDomainMapper;
    private final CashRequestWebMapper mapperTamp;

    @Override
    public CashRequestDTO getCashDemandById(UUID id) {
        try {
            return cashRequestDomainMapper.toDto(cashDemandRepository.getCashRequestById(id));
        } catch (Exception e) {
            log.error("Error retrieving cash request", e);
            throw new SessionServiceException("Failed to retrieve cash request : " + e.getMessage());
        }
    }

    @Override
    @Transactional
    public CashRequestDTO create(CashRequestDTO cashRequestDTO) {
        CashRequest cashRequest = mapperTamp.toDomain(cashRequestDTO);

        // Regle R1 : une demande doit contenir au moins une operation
        cashRequest.validateRequest();

        Map<String, BigDecimal> faceValuesMap = denominationRepository.findAllActiveDenominations()
                .stream()
                .collect(Collectors.toMap(
                        Denomination::getDenominationLabel,
                        Denomination::getFaceValue));

        // Regle R6 : validation de la ventilation par coupures
        cashRequest.getOperations().forEach(op -> op.validateAndCalculateTotal(faceValuesMap));

        return mapperTamp.toResponseDTO(cashDemandRepository.save(cashRequest));
    }

    @Override
    public CashRequestDTO update(UUID id, CashRequestDTO cashRequestDTO) {
        try {
            // FIX NPE : plus de .toString(), et null-safe (equals sur la constante)
            if ("CANCELLED".equals(cashRequestDTO.getRequestStatus())) {
                throw new IllegalStateException("Cannot update cancelled request");
            }
            CashRequest cashRequest = mapperTamp.toDomain(cashRequestDTO);
            return mapperTamp.toResponseDTO(cashDemandRepository.update(id, cashRequest));
        } catch (Exception e) {
            log.error("Error updating cash request", e);
            throw new SessionServiceException("Failed to update cash request : " + e.getMessage());
        }
    }

    @Override
    public List<CashRequestDTO> getAll() {
        try {
            return cashDemandRepository.findAll().stream()
                    .map(cashRequestDomainMapper::toDto)
                    .toList();
        } catch (Exception e) {
            log.error("Error getting all cash request", e);
            throw new SessionServiceException("Failed to get all cash request : " + e.getMessage());
        }
    }

    @Override
    public CashRequestDTO cancel(UUID id) {
        try {
            return mapperTamp.toResponseDTO(cashDemandRepository.cancelById(id));
        } catch (Exception e) {
            log.error("Error cancelling cash request", e);
            throw new SessionServiceException("Failed to cancel cash request : " + e.getMessage());
        }
    }

    @Override
    public CashRequestDTO cancelOperations(UUID id, List<CashOperationDTO> operations) {
        try {
            List<CashOperation> cashOperations = operations.stream()
                    .map(mapperTamp::toOperationDomain)
                    .toList();
            return mapperTamp.toResponseDTO(cashDemandRepository.cancelOperations(id, cashOperations));
        } catch (Exception e) {
            log.error("Error cancelling cash request", e);
            throw new SessionServiceException("Failed to cancel cash operations : " + e.getMessage());
        }
    }
}
Correctif 4 — NPE latent découvert pendant l'écriture des tests

CashRequestWebMapper — devise sans id dans toResponseDTO

Le problème : dans toResponseDTO, la ligne c.setId(detail.getCurrency().getId().getValue()) supposait que la devise avait toujours un id. Or quand l'arbre domaine est construit depuis un DTO (via toDomain), la devise ne porte que son isoCode — pas d'id ⇒ NullPointerException. C'est exactement le genre de bug que les tests aller-retour de mappers attrapent (le test B contient le cas toResponseDTO_deviseSansId_nePlantePas qui le verrouille).

Le correctif : l'id n'est copié que s'il existe. Imports inutiles nettoyés au passage. Tout le reste du fichier est identique à l'existant.

package com.bmci.workflow.fund.management.service.domain.mapperTemp1;

import com.bmci.q360.domain.entity.*;
import com.bmci.q360.domain.valueobjects.*;
import com.bmci.workflow.fund.management.service.domain.dto.*;
import org.springframework.stereotype.Component;

import java.util.ArrayList;
import java.util.List;
import java.util.stream.Collectors;

/**
 * @author Hamza Marouane
 * Correctif : dans toResponseDTO, l'acces detail.getCurrency().getId().getValue()
 * plantait en NullPointerException quand la devise n'avait pas d'id (cas d'un
 * arbre construit depuis un DTO, ou seul l'isoCode est renseigne). L'id est
 * desormais copie seulement s'il existe. Imports inutiles supprimes.
 */
@Component
public class CashRequestWebMapper {

    public CashRequest toDomain(CashRequestDTO dto) {
        if (dto == null) return null;
        CashRequest domain = new CashRequest();
        domain.setBranchCode(dto.getBranchCode());
        domain.setUidUser(dto.getUser());
        domain.setEstablishment(dto.getEstablishment());
        domain.setRequestStatus(dto.getRequestStatus());
        if (dto.getId() != null) {
            domain.setId(new CashRequestId(dto.getId()));
        }
        if (dto.getOperations() != null) {
            List<CashOperation> operations = dto.getOperations().stream()
                    .map(this::toOperationDomain)
                    .collect(Collectors.toList());
            domain.setOperations(operations);
        } else {
            domain.setOperations(new ArrayList<>());
        }

        return domain;
    }

    public CashOperation toOperationDomain(CashOperationDTO dtoOp) {
        if (dtoOp == null) return null;

        CashOperation domainOp = new CashOperation();
        if (dtoOp.getOperationType() != null) {
            domainOp.setOperationType(dtoOp.getOperationType());
        }
        if (dtoOp.getOperationSens() != null) {
            domainOp.setOperationSens(dtoOp.getOperationSens());
        }

        if (dtoOp.getId() != null) {
            domainOp.setId(new CashOperationId(dtoOp.getId()));
        }
        domainOp.setIsUrgent(dtoOp.getIsUrgent());
        domainOp.setOperationStatus(dtoOp.getOperationStatus());
        domainOp.setEstablishment(dtoOp.getEstablishment());
        if (dtoOp.getCurrencyDetails() != null) {
            List<OperationCurrencyDetail> currencyDetails = dtoOp.getCurrencyDetails().stream()
                    .map(dtoDetail -> {
                        OperationCurrencyDetail detail = new OperationCurrencyDetail();
                        if (dtoDetail.getId() != null) {
                            detail.setId(new OperationCurrencyDetailId(dtoDetail.getId()));
                        }
                        detail.setTotalAmountByCurrency(dtoDetail.getTotalAmountByCurrency());
                        Currency c = new Currency();
                        c.setIsoCode(dtoDetail.getCurrency().getIsoCode());
                        detail.setCurrency(c);
                        if (dtoDetail.getOperationDenominations() != null) {
                            List<OperationDenomination> denominations = dtoDetail.getOperationDenominations().stream()
                                    .map(dtoDenom -> {
                                        OperationDenomination od = new OperationDenomination();
                                        if (dtoDenom.getId() != null) {
                                            od.setId(new OperationDenominationId(dtoDenom.getId()));
                                        }
                                        od.setQuantity(dtoDenom.getQuantity());
                                        Denomination d = new Denomination();
                                        d.setDenominationLabel(dtoDenom.getDenomination().getDenominationLabel());
                                        od.setDenomination(d);
                                        return od;
                                    })
                                    .collect(Collectors.toList());
                            detail.setOperationDenominations(denominations);
                        }
                        return detail;
                    })
                    .collect(Collectors.toList());
            domainOp.setCurrencyDetails(currencyDetails);
        }

        return domainOp;
    }

    public CashRequestDTO toResponseDTO(CashRequest domain) {
        if (domain == null) return null;

        CashRequestDTO dto = new CashRequestDTO();
        if (domain.getId() != null) {
            dto.setId(domain.getId().getValue());
        }
        dto.setBranchCode(domain.getBranchCode());
        dto.setUser(domain.getUidUser());
        dto.setEstablishment(domain.getEstablishment());
        dto.setRequestStatus(domain.getRequestStatus());
        if (domain.getOperations() != null) {
            List<CashOperationDTO> ops = domain.getOperations().stream()
                    .map(op -> {
                        CashOperationDTO opDto = new CashOperationDTO();
                        opDto.setOperationSens(op.getOperationSens());

                        if (op.getId() != null) opDto.setId(op.getId().getValue());
                        opDto.setOperationType(op.getOperationType());
                        opDto.setOperationStatus(op.getOperationStatus());

                        opDto.setEstablishment(op.getEstablishment());
                        opDto.setIsUrgent(op.getIsUrgent());
                        opDto.setCashDeskNumber(op.getCashDeskNumber());
                        opDto.setCreatedAt(op.getCreatedAt());
                        opDto.setUpdatedAt(op.getUpdatedAt());
                        opDto.setInstitutionCode(op.getInstitutionCode());
                        opDto.setTargetCashDeskNumber(op.getTargetCashDeskNumber());

                        if (op.getCurrencyDetails() != null) {
                            var details = op.getCurrencyDetails().stream().map(detail -> {
                                var detailDto = new OperationCurrencyDetailDTO();
                                if (detail.getCurrency() != null) {
                                    CurrencyDTO c = new CurrencyDTO();
                                    c.setIsoCode(detail.getCurrency().getIsoCode());
                                    // FIX NPE : ne copier l'id que s'il existe
                                    if (detail.getCurrency().getId() != null) {
                                        c.setId(detail.getCurrency().getId().getValue());
                                    }
                                    detailDto.setCurrency(c);
                                }
                                detailDto.setTotalAmountByCurrency(detail.getTotalAmountByCurrency());

                                if (detail.getOperationDenominations() != null) {
                                    var denoms = detail.getOperationDenominations().stream().map(d -> {
                                        var dDto = new OperationDenominationDTO();
                                        DenominationDTO demDto = new DenominationDTO();
                                        demDto.setDenominationLabel(d.getDenomination().getDenominationLabel());
                                        dDto.setDenomination(demDto);
                                        dDto.setQuantity(d.getQuantity());
                                        return dDto;
                                    }).collect(Collectors.toList());
                                    detailDto.setOperationDenominations(denoms);
                                }
                                return detailDto;
                            }).collect(Collectors.toList());
                            opDto.setCurrencyDetails(details);
                        }
                        return opDto;
                    }).collect(Collectors.toList());
            dto.setOperations(ops);
        }

        return dto;
    }
}

        }
        domain.setItemType(entity.getItemType());
        domain.setFaceValue(entity.getFaceValue());
        domain.setDenominationLabel(entity.getDenominationLabel());
        return domain;

Lancer les tests

# 1. D'abord, tout compiler et installer les modules dans le repo Maven local

mvn clean install -DskipTests

# 2. Lancer les nouveaux tests, module par module :
mvn test -pl fund-management-domain/fund-management-domain-core          # Test A
mvn test -pl fund-management-domain/fund-management-application-service  # Test B
mvn test -pl fund-management-data-access -Dtest=CashRequestDataAccessMappersTest  # Test C

# (le -Dtest=... du module data-access evite de declencher d'anciens tests du module ;
#  tu peux aussi cibler une seule classe partout : -Dtest=CashOperationTest)

# 3. Verifier que les tests existants passent toujours :
mvn test -pl fund-management-application                                 # CashDemandControllerTest
mvn test -pl fund-management-data-access -Dtest=CashDemandRepositoryTest
Pourquoi pas mvn test à la racine ? Parce que ResourcePersistenceJpaRepositoryTest (module container) démarre tout le contexte Spring et exige la base PostgreSQL démarrée et configurée — il fera échouer le build sur une machine sans base. En attendant sa correction (Testcontainers, Sprint 2), on cible les modules. Si ta base locale tourne, mvn test à la racine fonctionne aussi.

Dans IntelliJ, plus simple encore : clic droit sur la classe de test → Run 'CashOperationTest'. Les trois classes sont des tests purs, ils s'exécutent instantanément.
